## Projet Architecture J2EE : Jeu de Réflexion
Ce projet consiste en la conception et le développement d'une application web de jeu de réflexion basée sur une architecture J2EE.
## 📝 Description du projet
L'objectif est de proposer une expérience de jeu fluide et interactive, où l'utilisateur peut s'authentifier et progresser à travers différents niveaux de difficulté. Le choix du jeu s'est porté sur un concept de réflexion original, excluant les jeux classiques de type Échecs, Dames, Sudoku ou 2048.
## Fonctionnalités principales

* Gestion des utilisateurs : Système d'authentification avec nom d'utilisateur et mot de passe.
* Niveaux de difficulté : Progression du joueur sur un minimum de trois niveaux distincts.
* Logique de jeu : Moteur de jeu géré côté serveur.

## 🛠 Stack Technique
Le projet repose exclusivement sur les technologies suivantes :

* Framework : Spring / Spring MVC
* Moteur de template : Thymeleaf
* Langage : Java

Note : Conformément aux spécifications, les frameworks Angular et Bootstrap ne sont pas utilisés dans ce projet.

## 🚀 Installation et Lancement

   1. Clonez le dépôt : git clone [URL_DU_DEPOT]
   2. Importez le projet dans votre IDE (IntelliJ IDEA ou Eclipse).
   3. Configurez le SDK Java (JDK 8 ou supérieur).
   4. Lancez l'application via le serveur Tomcat intégré ou la classe Main de Spring Boot.
   5. Accédez à l'application sur http://localhost:8080.
